CLI tool to check via node.js if you have a vulnerable bash [Shellshock (CVE-2014-6271)](http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2014-6271)

# Install

    npm install shellshock -g

# Execute

    shellshock

# Output

    ✗ vulnerable bash

or

    ✓ bash not vulnerable
